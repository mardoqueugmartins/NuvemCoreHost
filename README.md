#  NuvemCore Host - Responsive Hosting Landing Page

Projeto de estudo focado no desenvolvimento de uma landing page profissional para um serviço fictício de hospedagem, com ênfase em layout responsivo, UI components e boas práticas de front-end.

---

##  Tecnologias Utilizadas

- HTML5 (estrutura semântica)
- CSS3 (Flexbox, Grid, Animations, Media Queries)
- Figma (UI/UX Design)
- Git & GitHub (versionamento e documentação)

---

##  Objetivo do Projeto

Este projeto foi desenvolvido para praticar:

- Arquitetura de layout responsivo
- Estruturação semântica de páginas web
- Componentização visual (cards, pricing tables, badges)
- Microinterações e animações com CSS
- Design System (cores, tipografia, espaçamento)
- Conversão UI/UX (pricing, CTA, copywriting)
- Workflow de design → código (Figma → HTML/CSS)

---

##  Funcionalidades

- Hero section com destaque visual e glassmorphism
- Seção de benefícios com grid responsivo
- Tabela de preços com múltiplos planos
- Sistema de badges (Mais vendido / Recomendado)
- Formulário de busca de domínio
- Formulário de contato
- Totalmente responsivo (desktop, tablet e mobile)

---

##  Responsividade

O layout foi adaptado para:

- Desktop (≥ 1100px)
- Tablets (≤ 900px)
- Mobile (≤ 576px)

Com reorganização de componentes e ajustes de tipografia, imagens e grids.

---

##  Design

Todo o layout foi projetado previamente no **Figma**, seguindo princípios de:

- Hierarquia visual
- Espaçamento consistente
- Contraste e legibilidade
- UX orientado a conversão
